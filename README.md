# Coffee Shop Admin Portal
## Overview

Coffee Shop Admin Portal is a React-based Single Page Application (SPA) that allows administrators to manage coffee shop products. The app includes product management features such as adding, viewing, searching, updating, and deleting products.

## Features
- Responsive design
- Product listing page
- Add new products
- Edit product details
- Delete products
- Search functionality
- Client-side routing
- CRUD operations
- Custom React hooks
- Component testing

## Technologies Used
- React
- React Router DOM
- JavaScript
- CSS
- JSON Server
- Jest
- React Testing Library
- Vite

## Installation
git clone https://github.com/penzimbuthia-sudo/coffee-shop

cd coffee-shop

npm install

npm run server

npm run dev

useEffect
